#  User Authentication API

## Технологии:
- Node.js
- Express.js
- TypeScript
- ESLint + Prettier
- JSON Web Token (JWT)
- MVC-архитектура

# Установка
## Клонируйте репозиторий
 ```
https://github.com/yourbosss/10WEB.git
 ```
## Установите зависимости
 ```
yarn istall
 ```
---

## Запустите проект
 ```
yarn run dev
 ```
---
